Extension idéale pour accepter les paiements via M-Pesa, Airtel Money, Orange Money en utilisant PesaPay pour Woocommerce.

== Description ==

[Cette extension nécessite Woocommerce pour fonctionner].

PesaPay for Woocommerce est une extension qui vous permet d'intégrer un autre moyen de paiement pour votre boutique en ligne. 

En installant cette extension, vous ajoutez un Payment Gateway qui vous permet d'accepter et de recevoir les paiements partout dans le monde aux utilisateurs des Mobile Money qui n'ont pas de carte bancaire.

Installer et Activer cette extension pour ajouter les fonctionnalités suivantes à votre boutique:

* Paiement via M-Pesa
* Paiement via Airtel Money
* Paiement via Orange Money
* Fidéliser les clients n'ayant pas des cartes bancaires
* Profiter d'un compte Business gratuitement

Autres Avantages:

1. Les utilisateurs n'ont pas à quitter votre boutique pour payer.
1. Le paiement se fait dans quelques secondes.
1. Toutes les transactions sont sécurisées et bien cryptées.
1. Aucun frais vous sera facturé.

Accédez au site officiel de [PesaPay](https://pesapay.org/ "Votre portefeuille numérique") pour en savoir plus sur cette extension.


== Installation ==

Voici comment faire pour installer cette extension:


1. Télécharger le fichier de l'extension dans le dossier `/wp-content/plugins/pesapay-wp-plugin`, ou installer le plugin directement dans le répertoire de WordPress.
1. Aller dans le menu Extension de WordPress et activer l'extension.
1. Rendez-vous ensuite dans **PesaPay --> Configuration** pour configurer le plugin
1. Allez ensuite dans **Woocommerce --> Réglages --> Paiements**. 
1. Cochez PesaPay dans la liste pour activer l'extension.
